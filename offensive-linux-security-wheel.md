## وصف المحتوى

خريطة دائرية (Security Wheel) لأدوات لينكس المستخدمة من قبل فرق Red Team في محاكاة الهجمات الحقيقية وتقييم الضوابط الأمنية. الهدف هو الوعي الدفاعي وليس إساءة الاستخدام — فهم هذه الأدوات يساعد المدافعين على التعرف على التهديدات وتحصين الدفاعات.

> "This chart highlights Linux-based tools commonly used by red teams to emulate real-world attacks and evaluate security controls. The goal is awareness, not misuse."

## الأدوات مصنفة حسب الفئة

### 🔍 1. جمع المعلومات (Information Gathering)
| الأداة | الوصف |
|--------|-------|
| **theHarvester** | جمع رسائل البريد الإلكتروني والنطاقات الفرعية من المصادر العامة |
| **Amass** | تعداد النطاقات الفرعية ورسم خرائط أسطح الهجوم الخارجية (OWASP) |
| **Recon-ng** | إطار عمل استطلاع مفتوح المصدر بواجهة مشابهة لـ Metasploit |
| **DNSRecon** | فحص سجلات DNS واكتشاف النطاقات الفرعية ونقل المناطق |

### 💥 2. الاستغلال (Exploitation)
| الأداة | الوصف |
|--------|-------|
| **Metasploit** | أشهر إطار عمل للاستغلال والاختراق — آلاف الموديولات |
| **Pwntools** | مكتبة بايثون لكتابة استغلالات CTF وهندسة ثنائيات |
| **PTF** | PenTesters Framework — أداة تنظيم وإدارة أدوات الاختراق |
| **XSStrike** | كاشف ومستغل متقدم لثغرات XSS |
| **Exploit Pack** | مجموعة استغلالات جاهزة |
| **sqlmap** | أتمتة كشف واستغلال ثغرات SQL Injection |
| **Empire** | إطار عمل post-exploitation بوكلاء PowerShell/Python |
| **Impacket** | مكتبة بايثون للعمل مع بروتوكولات الشبكة (SMB, Kerberos...) |
| **Sliver** | إطار C2 (Command & Control) متعدد المنصات مفتوح المصدر |
| **Havoc** | إطار C2 حديث مع واجهة مستخدم رسومية |

### 🎯 3. فحص الثغرات (Vulnerability Scanning)
| الأداة | الوصف |
|--------|-------|
| **Nmap** | ماسح المنافذ والشبكات الأشهر — اكتشاف الخدمات وأنظمة التشغيل |
| **Nuclei** | ماسح ثغرات سريع يعتمد على قوالب YAML |
| **OpenVAS** | ماسح ثغرات شامل مفتوح المصدر (fork من Nessus) |
| **ZAP** | Zed Attack Proxy — ماسح ثغرات تطبيقات الويب من OWASP |
| **Nikto** | ماسح خوادم ويب — يكشف آلاف الملفات/CGIs الخطيرة |
| **WPScan** | ماسح متخصص لثغرات WordPress |
| **Burp Suite** | المنصة الأشهر لاختبار اختراق تطبيقات الويب |

### 🔫 4. هجمات الويب والتطبيقات
| الأداة | الوصف |
|--------|-------|
| **Wfuzz** | أداة fuzzing لتطبيقات الويب — اكتشاف directories/parameters |
| **Ffuf** | بديل سريع لـ Wfuzz مكتوب بلغة Go |

### 💀 5. ما بعد الاستغلال (Post-Exploitation)
| الأداة | الوصف |
|--------|-------|
| **Mimikatz** | استخراج كلمات المرور والـ hashes والتذاكر من ذاكرة ويندوز |
| **BloodHound** | رسم خرائط علاقات Active Directory لاكتشاف مسارات الهجوم |

### 📶 6. الهجمات اللاسلكية (Wireless Attacks)
| الأداة | الوصف |
|--------|-------|
| **Aircrack-ng** | مجموعة أدوات اختراق شبكات Wi-Fi (WEP/WPA/WPA2) |
| **Kismet** | كاشف شبكات لاسلكية ومحلل حزم |
| **Wifite** | أتمتة هجمات Wi-Fi — آلي بالكامل |
| **Reaver** | هجوم WPS PIN brute-force لاختراق شبكات Wi-Fi |
| **Bettercap** | هجمات MITM متكاملة — ARP spoofing, DNS spoofing, SSL stripping |

### 🔑 7. هجمات كلمات المرور (Password Attacks)
| الأداة | الوصف |
|--------|-------|
| **John the Ripper** | أسرع كاسر كلمات مرور — يدعم عشرات التنسيقات |
| **Hashcat** | أسرع كاسر hashes في العالم — يستخدم GPU |
| **Medusa** | أداة brute-force متوازية تدعم بروتوكولات متعددة |
| **Hydra** | أسرع أداة brute-force لتسجيل الدخول الشبكي |
| **CeWL** | توليد قوائم كلمات مخصصة من محتوى المواقع |
| **Patator** | أداة brute-force متعددة الأغراض مكتوبة ببايثون |

### 📱 8. أمن الموبايل (Mobile Security)
| الأداة | الوصف |
|--------|-------|
| **Androguard** | هندسة عكسية وتحليل تطبيقات أندرويد |
| **MobSF** | Mobile Security Framework — تحليل ثغرات تطبيقات الموبايل آلياً |
| **Kali NetHunter** | منصة اختبار اختراق للأجهزة المحمولة (أندرويد) |

### 🎭 9. الهندسة الاجتماعية (Social Engineering)
| الأداة | الوصف |
|--------|-------|
| **SET** | Social Engineering Toolkit — أتمتة هجمات التصيد والهندسة الاجتماعية |
| **Gophish** | منصة محاكاة التصيد لتدريب المؤسسات |
| **King Phisher** | أداة حملات تصيد متكاملة |

### 🌐 10. هجمات الشبكات (Network-based Attacks)
| الأداة | الوصف |
|--------|-------|
| **Responder** | اعتراض تجزئات NTLM وطلبات LLMNR/NBT-NS في الشبكات الداخلية |
| **Yersinia** | هجمات بروتوكولات الطبقة الثانية (STP, CDP, DTP, VTP...) |
| **Scapy** | مكتبة بايثون لتوليد وتحليل وتعديل حزم الشبكة |
| **Netcat** | سكين الجيش السويسري للشبكات — TCP/UDP connections, port scanning |

## الدروس المستفادة

1. **تغطية شاملة لدورة حياة الهجوم:** الأداة تغطي جميع مراحل Kill Chain — من جمع المعلومات حتى post-exploitation
2. **أدوات لينكس الأساسية لـ Red Team:** خريطة مرجعية ممتازة لأي مختبر اختراق
3. **ثغرات الأدوات الحديثة:** غياب أدوات AI-based (مُلاحظ في التعليقات) — فرصة للبحث والتطوير
4. **قيمة تعليمية:** يمكن استخدامها كمنهج تدريبي لتغطية أدوات offensive security على لينكس

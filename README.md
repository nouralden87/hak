### أدوات التحقيق الرقمي (Forensics)

- **Autopsy** — منصة تحقيق رقمي متكاملة، استرجاع ملفات محذوفة، استخراج metadata
- **Wireshark** — أشهر محلل شبكات، فحص الحزم وكشف الاتصالات المشبوهة
- **Bulk Extractor** — استخراج إيميلات، URLs، بطاقات ائتمان من الأقراص بدون تحليل نظام الملفات
- **Guymager** — استنساخ أقراص وأخذ صور جنائية (Forensic Imaging)
- **Toolsley** — 10+ أدوات ويب: توقيع الملفات، فحص الهاش، مفتش ثنائي
- **Volatility 3** — أقوى إطار لتحليل الذاكرة (Memory Forensics)
- **Binwalk** — تحليل وفك البرامج الثابتة (Firmware Analysis)
- **pspy** — مراقبة عمليات لينكس بدون صلاحيات root
